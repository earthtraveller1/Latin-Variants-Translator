What is Egg Latin?


Egg Latin is a part of the joke latin languages, in which every vowel in a syllable precedes with the word Egg.

For example, Hello would be Heggelleggo (H[egg]ell[egg]o) which has 2 syllables with a vowel in each of them, which makes 2 eggs.

Of course you wouldn't add another egg preceding the 'e' in an egg you just added, because that would just make an infinite loop.

(The egg in H[egg]elleggo would never be H[eggegg]elleggo).

However, when the english sentence contains the word Egg in the original sentence, you would still add an egg behind it, making it eggegg.

If there's a word like Tea, which has 2 vowels in 1 syllable, you would only add egg to the first one, making Teggea instead of Teggeegga.
